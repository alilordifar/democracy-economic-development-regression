This project analyzes the relationship between democracy and economic development using country-level data from the World Bank and the V-Dem Institute. The primary outcome variable is GDP per capita (constant USD), used as a proxy for average living standards, while democracy is measured using the Liberal Democracy Index. The analysis incorporates additional socioeconomic controls such as population, education, life expectancy, trade, and other macroeconomic indicators to account for potential confounding factors. The goal of this project is to build and interpret regression models, evaluate key assumptions, and perform residual analysis to better understand how democratic institutions are statistically associated with economic outcomes across countries over time.
